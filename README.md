# ShadowLearn AI

ShadowLearn AI is an AI-powered study assistant that transforms PDF notes into interactive learning resources.

## Features

* PDF Text Extraction
* AI Quiz Generator
* Flashcard Generator
* Study Plan Generator
* Ask Notes AI Tutor
* Exam Booster
* Viva Question Generator

## Tech Stack

* Python
* Streamlit
* Google Gemini API
* PyPDF2
